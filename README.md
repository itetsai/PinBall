# PinBall
- Unity 版本: 2017.3.1f1

## 操作說明
- 空白鍵:發射彈珠
- Z:左側擋板
- M:右側擋板
- F1:位移彈珠至滑鼠位置
## 機關設計
- 藍色:彈射板(雷電效果)
- 紅色:彈射板(火花效果)
- 紫色:彈射桿
- 白色:吸力
- 黑色:斥力，將球打至中間可多加一顆球
